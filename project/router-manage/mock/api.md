# api文档

## 登录接口
  - **接口协议**：http post

  - **接口地址**：/api/login

  - **接口参数**
    | 编号 | 字段     | 类型   | 是否必填 | 描述                                                    |
    | :--: | -------- | ------ | :------: | ------------------------------------------------------- |
    |  1   | userName | String |    M     | 用户名                                                  |
    |  2   | passWord | String |    M     | 密码                                                    |
    |  3   | sign     | String |    M     | MD5签名(userName+ ‘zxr’+ passWord)  ，huashu是约定默认值 |
    
  - **返回参数**
    | 编号 | 字段    | 类型   | 描述                       |
    | :--: | ------- | ------ | -------------------------- |
    |  1   | code    | int    | 200 接口正常返回，否则异常 |
    |  2   | message | String | 错误描述信息               |
    |  3   | data    | json   | 返回数据                   |
    
  - **接口返回数据**
    ```json
      {
        "code": 200,
        "message": "SUCCESS",
        "data": {  } 
      }
    ```
