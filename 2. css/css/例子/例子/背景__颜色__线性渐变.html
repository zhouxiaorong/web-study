背景颜色-线性渐变 : 向下/向上/向左/向右/对角方向

渐变背景

CSS 语法 background-image: linear-gradient(direction, color-stop1, color-stop2, ...);

定义与用法linear-gradient() 函数用于创建一个线性渐变的 "图像"。

为了创建一个线性渐变，你需要设置一个起始点和一个方向（指定为一个角度）的渐变效果。

你还要定义终止色。终止色就是你想让Gecko去平滑的过渡，并且你必须指定至少两种，当然也会可以指定更多的颜色去创建更复杂的渐变效果。

值 实例 描述
direction to top, to right, to bottom, to left, to top left, to bottom right 用角度值指定渐变的方向（或角度）。
color-stop1, color-stop2,... red, green, yellow, rgba(255,0,0,.3) 用于指定渐变的起止颜色。
